## virgo-plus

My modification of virgo (<https://github.com/henkman/virgo>) for windows virtual desktop managment.
Modify shotcut keys according to Linux desktops, and enable go/move/move-and-go to previous/next desktop.

| Shotcut Key            | Function                                          |
| ---------------------- | ------------------------------------------------- |
| Ctrl+Alt+1/2/3/4       | go to desktop 1/2/3/4                             |
| Ctrl+Alt+Left          | go to previous desktop                            |
| Ctrl+Alt+Rgiht         | go to next desktop                                |
| Ctrl+Shift+1/2/3/4     | move active window to desktop 1/2/3/4             |
| Ctrl+Shift+Left        | move active window to previous desktop            |
| Ctrl+Shift+Rgiht       | move active window to next desktop                |
| Ctrl+Shift+Alt+1/2/3/4 | move and go to desktop 1/2/3/4                 |
| Ctrl+Shift+Alt+Left    | move and go to previous desktop                   |
| Ctrl+Shift+Alt+Rgiht   | move and go to next desktop                       |
| Ctrl+Shift+Alt+S       | enable/disable shotcut keys of virgo-plus         |
| Ctrl+Shift+Alt+Q       | exits the virgo-plus program                      |
