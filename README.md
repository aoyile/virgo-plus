
## virgo-plus

My modification of virgo (<https://github.com/henkman/virgo>) for windows virtual workspaces:
  + modify shortcut keys according to Linux workspaces management.
  + enable shortcut keys for go/move/move-and-go to previous/next workspace.

| Shortcut Key           | Function                                  |
| ---------------------- | ----------------------------------------- |
| Alt+1/2/3/4            | go to workspace 1/2/3/4                   |
| Ctrl+1/2/3/4           | move active window to workspace 1/2/3/4   |
| Ctrl+Shift+Alt+1/2/3/4 | move and go to workspace 1/2/3/4          |
| Ctrl+Shift+Alt+S       | enable/disable shotcut keys of virgo-plus |
| Ctrl+Shift+Alt+Q       | exits the virgo-plus program              |
| Ctrl+Alt+Left          | go to previous workspace                  |
| Ctrl+Alt+Right         | go to next workspace                      |
| Shift+Alt+Left         | move active window to previous workspace  |
| Shift+Alt+Right        | move active window to next workspace      |
| Ctrl+Shift+Alt+Left    | move and go to previous workspace         |
| Ctrl+Shift+Alt+Right   | move and go to next workspace             |
